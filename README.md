# MVVMHttpManager
MVVM模式网络请求框架
# 使用方式  
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```  
```
dependencies {
	        implementation 'com.github.freakcsh:MVVMHttpManager:1.0.0'
	}
```  
具体使用介绍请查看博文：  
 [基于okhttp3+rxjava2+retrofit2的MVVM模式网络请求](https://blog.csdn.net/freak_csh/article/details/90484863)
